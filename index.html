Política de Privacidade — InfoDriver

Última atualização: 08 de março de 2026

O aplicativo InfoDriver foi desenvolvido para auxiliar motoristas parceiros das plataformas Uber e 99. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários.

1. Informações que coletamos

  • Localização precisa (GPS): Utilizada em tempo real para calcular rotas e distâncias de corridas. Não é armazenada em servidores externos.
  • Endereço de e-mail: Coletado durante o cadastro/login via Firebase Authentication, usado exclusivamente para autenticação.
  • Dados de uso do aplicativo: O app analisa as informações de corridas exibidas na tela do dispositivo (via Serviço de Acessibilidade) para fornecer decisões de aceite/rejeição. Esses dados são processados localmente.

2. Como usamos as informações

  • Autenticar o usuário e verificar o status de assinatura Premium.
  • Calcular métricas de corrida (R$/km, R$/hora, trânsito) em tempo real.
  • Consultar rotas via API do Google Maps para enriquecer as informações de corrida.

3. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros, exceto:

  • Firebase (Google): Autenticação e verificação de assinatura.
  • Google Maps API: Endereços de destino consultados para cálculo de rota.

4. Serviço de Acessibilidade

O InfoDriver utiliza a API de Acessibilidade do Android para capturar informações exibidas na tela dos aplicativos Uber Driver e 99 Driver. Essas informações são processadas exclusivamente no dispositivo do usuário.

5. Localização

A localização GPS é acessada em segundo plano enquanto o serviço Copiloto estiver ativo. O usuário pode desativar o serviço a qualquer momento pelo botão principal do aplicativo.

6. Retenção de dados

As informações de autenticação são mantidas enquanto a conta estiver ativa. O usuário pode solicitar a exclusão através da opção disponível nas configurações do aplicativo.

7. Menores de idade

O InfoDriver é destinado exclusivamente a adultos (motoristas de aplicativo). Não coletamos intencionalmente dados de menores de 18 anos.
